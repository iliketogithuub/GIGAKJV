## SKJV Web App (skjvapp.com / skjv.base44.app)

**Description:** A refined, distraction-free Bible reading experience featuring the classic King James Version alongside a modern-vocabulary adaptation for deep, accessible study.

### App Features / Pages
- **Bible Reader** -- main reading interface (KJV + SKJV side by side)
- **Bible Search**
- **Core Biblical Doctrines**
- **My Bookmarks**
- **My Notes**
- **My Profile**
- **Prayer Journal**
- **Reading Plans**
- **Sermon Builder**
- **Sermon Reader**
- **Study Tools**
- **Admin Analytics**
- **Donate Page**

### Notes on Access
The app appears to be a modern web application (SPA). It is **not clear whether the full SKJV text can be easily downloaded or scraped** from this site; it is likely served dynamically and may require user account access for full functionality. It is designed as a reading/study tool rather than a text distribution platform.
